# Get started with the Model Context Protocol (MCP)

## Start

* Only Server
    ```bash
    uv run server.py
    ```

* Client using Server
  
  server.py 를 사용하는 function call client 예시
    ```bash
    uv run client-chat.py server.py
    ```
